# Contributing to play597 Projects

play597 프로젝트에 기여해 주셔서 감사합니다!

## How to Contribute

### 1. Issue 확인

- 작업 전 관련 이슈가 있는지 확인해주세요
- 새로운 기능이나 버그 수정은 먼저 이슈를 생성해주세요

### 2. Fork & Clone

```bash
git clone https://github.com/play597/<repository>.git
cd <repository>
```

### 3. Branch 생성

```bash
git checkout -b feature/your-feature-name
# 또는
git checkout -b fix/bug-description
```

**Branch 네이밍 규칙:**
- `feature/` - 새로운 기능
- `fix/` - 버그 수정
- `docs/` - 문서 수정
- `refactor/` - 리팩토링

### 4. 변경사항 커밋

```bash
git add .
git commit -m "feat: add new feature"
```

**커밋 메시지 규칙:**
- `feat:` - 새로운 기능
- `fix:` - 버그 수정
- `docs:` - 문서 수정
- `style:` - 코드 스타일 변경
- `refactor:` - 리팩토링
- `test:` - 테스트 추가/수정
- `chore:` - 빌드, 설정 등

### 5. Pull Request

- PR 템플릿을 따라 작성해주세요
- 리뷰어 지정 후 리뷰를 기다려주세요

## Code Style

- 각 프로젝트의 linter/formatter 설정을 따라주세요
- PR 전 테스트가 통과하는지 확인해주세요

## Questions?

궁금한 점이 있으면 이슈로 문의해주세요.
