# Security Policy

## Supported Versions

현재 보안 업데이트가 지원되는 버전:

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

보안 취약점을 발견하셨다면:

### 1. 비공개로 신고해주세요

**공개 이슈로 보안 취약점을 신고하지 마세요.**

대신 다음 방법을 이용해주세요:
- GitHub Security Advisory 사용
- 또는 관리자에게 직접 연락

### 2. 포함할 정보

- 취약점 유형 및 위치
- 재현 단계
- 영향 범위
- 가능하다면 수정 제안

### 3. 응답 시간

- 신고 확인: 48시간 이내
- 초기 평가: 7일 이내
- 수정 릴리스: 심각도에 따라 결정

## Security Best Practices

프로젝트 기여 시:

- 민감한 정보(API 키, 비밀번호 등)를 커밋하지 마세요
- 의존성을 최신 상태로 유지하세요
- 보안 관련 린터 경고를 무시하지 마세요
